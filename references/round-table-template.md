# 圆桌讨论模板

## 梦蝶讨论标准格式

### 议题部分

```markdown
## 议题

[本次讨论的核心问题]

例如：「AI是否会取代艺术家？」
```

### 参与者列表

```markdown
## 参与者列表

| 参与者 | Skill来源 | 角色说明 |
|--------|----------|---------|
| 马斯克 | elon-musk-perspective | 技术疯子、SpaceX/Tesla创始人 |
| 乔布斯 | jobs-perspective | 苹果创始人，极简主义者 |
| 库克 | tim-cook-perspective | 苹果CEO，供应链大师 |
| 川普 | trump-perspective | 前美国总统，商业谈判高手 |
```

### 讨论规则

```markdown
### 轮次设计

1. **开场**：每个参与者用1-2句话亮出基本立场
2. **第一轮**：深度讨论，每人有3-5句话展开
3. **第二轮**：回应其他人的观点，推进讨论
4. **自由讨论**：根据讨论进展自然推进
5. **收尾**：主持人综合，形成共识结论 + 保留分歧点

### 主持人规则

主持人负责：
- 维持讨论秩序
- 确保每个参与者都有发言机会
- 追问关键问题
- 最终综合形成结论

### 发言规则

- 每轮发言不超过5句话
- 被打断时主持人介入
- 允许「跳过」某轮的发言
```

## 结论格式

### 共识结论

```markdown
### 共识结论

[主持人在综合所有人观点后形成的结论，1-2句话]
```

### 多元观点阵列

```markdown
### 多元观点阵列

| 参与者 | 核心观点 | 与其他参与者的分歧 |
|--------|---------|-----------------|
| 马斯克 | [观点] | vs 乔布斯在[某点]上分歧：... |
| 乔布斯 | [观点] | vs 库克在[某点]上分歧：... |
| 库克 | [观点] | vs 川普在[某点]上分歧：... |
| 川普 | [观点] | vs 马斯克在[某点]上分歧：... |
```

### 保留分歧点

```markdown
### 保留分歧点

1. [分歧1]：...
2. [分歧2]：...
3. [分歧3]：...
```

## 快速启动模板

当用户提供完整信息时，直接使用以下格式启动：

```markdown
# 梦蝶讨论

**日期**：[日期]
**议题**：[议题]

## 参与者

- [角色1]（[Skill来源]）- [角色说明]
- [角色2]（[Skill来源]）- [角色说明]

**主持人**：默认

## 讨论过程

### 开场陈述

[角色1]：「[1-2句话基本立场]」

[角色2]：「[1-2句话基本立场]」

### 第一轮

主持人：「请[角色1]先展开说说。」

[角色1]：「[3-5句话深度展开]」

...
```

## 部分信息时的引导模板

当用户只提供部分信息时，需要追问：

### 只提供议题

```
梦蝶：「好的，你想讨论『[议题]』。请告诉我参与者是谁？有几个选项：

1. 直接告诉我参与者：「让马斯克和乔布斯讨论」
2. 描述需求我帮你匹配：「需要一个技术视角和一个商业视角」

或者直接说「帮我选」，我根据议题自动匹配。」
```

### 只提供参与者

```
梦蝶：「好的，你想让[角色1]和[角色2]讨论。请告诉我议题是什么？

可以直接说议题，或者「帮我选议题」，我根据参与者推荐。」
```

### 信息不足

```
梦蝶：「想要开讨论，需要两件事：
1. 参与者（谁参加讨论？）
2. 议题（讨论什么？）

请告诉我至少其中一个，另一个我来帮你补充。」
```

## 特殊场景模板

### 继续上次讨论

```markdown
# 梦蝶讨论（继续）

**日期**：[新日期]
**议题**：[原议题]
**接续**：上一次讨论于[日期]进行

## 上次结论回顾

【共识结论】
[原结论]

【保留分歧点】
[原分歧]

## 本次继续

主持人：「上次讨论结论是...[总结]。今天我们继续，有没有人想改变立场或补充新观点？」
```

### 换参与者

```markdown
# 梦蝶讨论（换人）

**日期**：[新日期]
**议题**：[原议题]
**变更**：替换[原角色]为[新角色]

主持人：「今天[新角色]加入讨论，代替之前的[原角色]。请[新角色]先说说你的基本立场。」
```

### 换议题

```markdown
# 梦蝶讨论（新议题）

**日期**：[新日期]
**新议题**：[新议题]
**旧议题**：[旧议题]（已结束）

主持人：「之前的讨论结束了。现在我们换一个新议题：[新议题]。请各位先说说对新议题的基本立场。」
```

## 主持人发言模板

### 开场

```
主持人：「各位好，今天我们讨论的议题是：[议题]。请每位用1-2句话说说你们的基本立场。」
```

### 引导发言

```
主持人：「好，各位都亮明了基本立场。现在我们来深入聊聊。请[角色1]先展开说说。」
```

### 追问

```
主持人：「[角色1]，你刚才说[观点]，能具体说说你的依据是什么？」
```

### 拉回主题

```
主持人：「好，[角色1]提出了一个有趣的观点。不过我们的议题是[议题]，请各位围绕这个核心问题继续讨论。」
```

### 总结过渡

```
主持人：「好，讨论得很充分了。让我来总结一下今天的讨论。」
```
